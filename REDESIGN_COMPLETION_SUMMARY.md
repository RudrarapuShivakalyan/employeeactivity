## 🎓 Learning Resources

### For Further Customization:

1. **CSS Variables** - Modify `:root` in `src/index.css`
2. **Component Props** - Check `src/components/common.jsx` for all options
3. **Colors** - Update color values in CSS variables
4. **Typography** - Adjust font sizes and weights
5. **Animations** - Modify `@keyframes` in `src/index.css`

### Example: Changing Primary Color
```css
/* In src/index.css, update: */
--color-primary: #4f46e5;        /* Change to your color */
--color-primary-light: #6366f1;
--color-primary-dark: #4338ca;
--color-primary-50: #eef2ff;
```

## 📞 Support & Help

### Common Issues:

**Q: Components not showing up?**
A: Make sure to import from `src/components/common.jsx`

**Q: Styling not working?**
A: Ensure Tailwind CSS is properly configured in `vite.config.js`

**Q: API not connecting?**
A: Check backend is running on `localhost:5000` and tokens are in localStorage

**Q: Animations too slow/fast?**
A: Adjust animation duration in CSS classes (e.g., `duration-200`, `duration-500`)
